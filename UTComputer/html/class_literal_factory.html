Literal *  target 
)

Permet d'ajouter une variable pointant sur une autre littérale.

Provoque une erreur si le nom de variable correspond à un nom de fonction ou d'opérateur. Provoque une erreur si le nom de variable correspond à un nom de variable déjà utilisé. Provoque une erreur si le nom de variable n'est pas valide (suite de chiffre ou de lettres majuscules, commençant par une lettre majuscule).

Paramètres
atom: Nom de la variable
target: Pointeur sur la littérale à pointer
Renvoie
Literal & LiteralFactory::addLiteralFromString ( const QString &  exp)

Permet d'ajouter une litérale à partir de la forme qu'elle devrait avoir une fois la fonction toString() appliquée.

Exemple : "1$2" permet de créer la litérale complexe 1$2 Déclenche une erreur s'il est impossible de créer une littérale à partir de la chaine de caractères.

Paramètres
exp: Chaine de caractères à convertir en litérale
Renvoie
La litérale créée
