Description détaillée

Classe permettant de gérer les nombres complexes. Composée de deux Number.

Un complexe ayant une partie imaginaire nulle se comporte comme un "réel".

Documentation des constructeurs et destructeur

const Number &  re = 0,
const Number &  im = 0 
)
inlineprivate

Construit une littérale complexe à partir de deux littérale "réelles" (entières, rationnelles ou réelles). Si la partie imaginaire n'est pas précisée, elle vaut 0.

Paramètres
m: Pointeur sur la Factory qui a créé le complexe
re: La partie réelle
im: La partie imaginaire
